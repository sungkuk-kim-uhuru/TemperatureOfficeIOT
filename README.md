# Express Generator with Vue-CLI


開発する時
```
git clone https://github.com/sungkuk-kim-uhuru/TemperatureOfficeIOT.git
cd backend 
npm install 
cd frontend
npm install
npm run dev
```

運営する時
```
git clone https://github.com/sungkuk-kim-uhuru/TemperatureOfficeIOT.git
cd backend 
npm install 
cd frontend
npm install
npm run build
cd ..
node app.js
```
